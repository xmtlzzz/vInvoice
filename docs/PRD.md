# vInvoice项目PRD
## 必须的内容
1. 报销费用管理
   - 项目整体的报销费用计算
   - 项目需要的发票总数
   - 费用是否已经报销（已经报销可以☑️反之❌️）
   - 报销费用的类别（比如交通、住宿、个人产生等，需要设置预设分类）
   - 项目费用是否已经提交报销

2. 月度统计
    - 显示当月产生的报销费用总集

3. 发票pdf管理
   - 支持发票文档的上传（通常是pdf格式）
   - 上传的发票按照归属项目目录进行存放
   - 格式化发票文件名（上传序号-发票金额-日期-费用类型.pdf）

## 后续扩展内容
1. 用户管理
2. 图标展示

## 页面设计
1. 首页：展示所有的项目
2. 项目子页：按照日期归档，每个条目内存放对应时间录入的费用信息
3. 统计页面：统计每个月的费用支出
